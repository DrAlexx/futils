Detailed Description

The binary tree is a famous data structure that has been described many times. As example you can find the description here Wiki - Binary tree.

Guess you know such standart C++ classes as std::set and std::map. They are ussualy implemented as Wiki - Red-black tree. This implementaions is similar to STL but have some differences:

  • Here is only one class namely binary_tree that is a map and the set. if you want to have a set you should pass template parameter T as void. as example:
    binary_tree::binary_tree<int, void> my_set; //this is a set that can store int keys only
    binary_tree::binary_tree<int, std::string> my_map; //this is a map where key has type int and maped value is std::string
  • This implementaion uses only two links per node. It helps to reduce memory usage and I hope it helps to improve performance. But without a link to the parent node impossible to implement iterator based acces. So it isn't present. Instead you can use enumerators.
  • balancer type can be customized via template parameter B. But now only avl_balancer is implemented.

avl_balancer provides AVL tree. Most implementations of AVL tree are recursive. But this library contains iterative inserting and removing.
